# Carousel jQuery

En este ejercicio usamos jQuery para dar funcionalidad a un carrusel de imágenes, así mismo utilizamos Bootstrap como framework de CSS.

### ¿Qué es jQuery?
>jQuery is a JavaScript Library.

>jQuery greatly simplifies JavaScript programming.

### ¿Qué es Bootstrap?

>Bootstrap is the most popular HTML, CSS, and JS framework for developing responsive, mobile first projects on the web.

Es importante recordar que para ejecutar el código cuando la página ha sido cargada hemos incluido lo siguiente:

    $(document).ready(cargarPagina);

Y en el documento HTML, incluimos los siguientes scripts:

    <script src="https://ajax.googleapis.com/ajax/libs/jquery/1.12.4/jquery.min.js"></script>
	<script src="js/bootstrap.min.js"></script>

Así como:

    <link href="css/bootstrap.min.css" rel="stylesheet">

En nuestra etiqueta head, esto para poder trabajar con Bootstrap.
